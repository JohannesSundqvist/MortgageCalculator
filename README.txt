Requirements for launching this application:
- Your favorite IDE, supporting Java
- Git Shell / Git CMD

Steps for launching this application:
1. Clone this repository (git clone https://github.com/JohannesSundqvist/MortgageCalculator)
2. Launch your favorite IDE, and open the MortgageCalculator project
3. Run the project 

If a runnable/main CLASS has to be selected, the path is \MortagePlan\src\main\java\Main.java

If a runnable/main METHOD has to be selected, it is main(String[] args)
